# GigFlow

GigFlow is a full-stack freelance marketplace web application where users can post gigs, browse available gigs, place bids, and manage their profiles.

---

## Features

- User authentication (JWT)
- Create and view gigs
- Bid on gigs
- View posted gigs
- View placed bids
- Protected routes
- User profile management

---

## Tech Stack

**Frontend**
- React
- Redux Toolkit
- React Router
- Tailwind CSS

**Backend**
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- bcrypt

---


Setup Instructions ----

Backend --

cd server
npm install
npm run dev

Frontend --

npm install
npm start
